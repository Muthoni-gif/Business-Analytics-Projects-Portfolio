6 Queries — From Basic to Advanced

Each query builds on the last — starting with overall metrics and progressing to window functions and multi-dimensional CTE analysis.

Q01

Overall Churn Rate

Aggregation · CASE 
SELECT
  COUNT(*) AS total_customers,
  SUM(CASE WHEN "Churn" = 'Yes' THEN 1 ELSE 0 END) AS churned,
  ROUND(SUM(CASE WHEN "Churn" = 'Yes' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS churn_rate
FROM telco_churn;
// Results
total_customerschurnedchurn_rate
7043186926.54%
💡 Over 1 in 4 customers is leaving — a serious business problem

GROUP BY · ORDER BY

Q03

Churn by Tenure Group (Cohort Analysis)

CASE · Cohort 
SELECT
  CASE
    WHEN "tenure" <= 12 THEN '0-12 months'
    WHEN "tenure" <= 24 THEN '13-24 months'
    WHEN "tenure" <= 48 THEN '25-48 months'
    ELSE '49+ months'
  END AS tenure_group,
  COUNT(*) AS total_customers,
  ROUND(SUM(CASE WHEN "Churn" = 'Yes' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS churn_rate
FROM telco_churn
GROUP BY tenure_group
ORDER BY churn_rate DESC;
// Results

Q04

Churn by Internet Service & Avg Charges

AVG · GROUP BY 
SELECT
  "InternetService",
  COUNT(*) AS total_customers,
  ROUND(SUM(CASE WHEN "Churn" = 'Yes' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS churn_rate,
  ROUND(AVG("MonthlyCharges")::numeric, 2) AS avg_monthly_charges
FROM telco_churn
GROUP BY "InternetService"
ORDER BY churn_rate DESC;

Q05

High-Risk Segments — CTE Analysis

CTE · WITH clause 
WITH churn_summary AS (
  SELECT
    "Contract", "InternetService",
    COUNT(*) AS total_customers,
    ROUND(SUM(CASE WHEN "Churn" = 'Yes' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS churn_rate,
    ROUND(AVG("MonthlyCharges")::numeric, 2) AS avg_charges
  FROM telco_churn
  GROUP BY "Contract", "InternetService"
)
SELECT * FROM churn_summary
WHERE churn_rate > 30
ORDER BY churn_rate DESC;
// Results
ContractInternetServicetotal_customerschurn_rateavg_charges
Month-to-monthFiber optic212854.61%7.02
Month-to-monthDSL122332.22%0.22
💡 Month-to-month fiber optic customers — 2,128 people paying 7/month — churn at 54.61%
Q06

Window Function — Avg Charges by Partition

Window · PARTITION BY 
SELECT
  "Contract", "tenure", "MonthlyCharges", "Churn",
  AVG("MonthlyCharges"::numeric)
    OVER (PARTITION BY "Contract") AS avg_charges_by_contract,
  AVG("MonthlyCharges"::numeric)
    OVER (PARTITION BY "InternetService") AS avg_charges_by_internet,
  COUNT(*) OVER (PARTITION BY "Contract") AS total_in_contract
FROM telco_churn
WHERE "Churn" = 'Yes'
LIMIT 5;

Key Findings

What the Data Reveals

Four actionable insights for the retention team — each backed by SQL evidence.

⚠ Highest Risk Segment
54.61%

Month-to-month fiber optic customers churn at over 54% — more than half. With 2,128 customers paying 7/month, this represents significant revenue at risk.

📅 Critical Window
Year 1

Nearly 47% of customers leave in year 1. After 4 years the rate drops to 9.5%. The onboarding experience is the most important retention lever.

📋 Contract Impact
15×

Month-to-month customers churn at 15 times the rate of two-year contract customers (42.71% vs 2.83%). Moving customers to longer contracts is the single biggest retention lever.

💰 Price-Value Gap
1.50

Fiber optic customers pay 1.50/month — 57% more than DSL — yet churn at 41.89%. They don't feel the premium service justifies the price.
